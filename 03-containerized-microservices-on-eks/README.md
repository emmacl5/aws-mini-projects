# Project 03: Containerized Microservices on Amazon EKS

## 📌 Project Status

🚧 In Progress

## 📖 Project Overview

This project demonstrates how to containerize, deploy, scale, and operate a microservices application using Docker, Amazon ECR, Kubernetes, and Amazon EKS.

The application consists of two independently deployable Python services. Each service is packaged as a Docker image, stored in Amazon ECR, and deployed to an Amazon EKS cluster using Kubernetes manifests.

The project focuses on container orchestration, service-to-service communication, health checks, scalability, observability, security, and repeatable deployments.
